# Affiliates Logo Files

Place your affiliate logo files here in AVIF format.

## Expected Files

- `affiliate-1.avif` - First affiliate logo
- `affiliate-2.avif` - Second affiliate logo
- `affiliate-3.avif` - Third affiliate logo
- `affiliate-4.avif` - Fourth affiliate logo
- Add more as needed

## Image Guidelines

- **Format**: AVIF (modern, highly optimized format)
- **Size**: Logos should be roughly square (1:1 aspect ratio)
- **Resolution**: 400x400px minimum
- **Background**: Transparent or white preferred
- **File size**: Keep under 50KB for optimal performance

## Converting to AVIF

If you have PNG or JPG logos, you can convert them to AVIF using:
- Online: https://avif.io
- Command line: `npx @squoosh/cli --avif auto yourlogo.png`
